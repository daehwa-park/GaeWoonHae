# 게운해 (게임으로 운동 해)
화상 연결과 게임을 통한 홈 트레이닝 웹사이트

# 프로젝트 사용 방법
- 테스트 할 내용은 branch를 새로 만들어 작성
- 새로운 기능 merge시 메인 README에 해당 내용 업데이트
- swagger: http://localhost:5000/swagger-ui/index.html#/

# React
### 실행
npm run start

# Spring

### version
- SDK : Azur zulu java 17
- Spring : 3.1.1
- build tool : Gradle groovy


### Dependency
- Spring Web
- Spring Data JPA
- Lombok
- Mysql Driver
- Spring Devtools
